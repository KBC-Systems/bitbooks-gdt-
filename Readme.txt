DAPP: bitbooks-gdt-

Installation :

1. Go to your own lisk server. If you do have go to sever URL: http://149.56.46.111:7000/ or liks testnet server.

2. Create New Account if you do not have one and login to server using your secret passphrase.

3. Go to Application Tab > Click on launched application bitbooks in category 'Education'.

4. Yo will be redirected to default bitbooks dApplication.
